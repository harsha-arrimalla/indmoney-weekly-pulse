# INDMoney Weekly Pulse

Automated weekly analysis of Play Store reviews using Google Gemini AI. Transforms raw feedback into high-impact product insights.

## Features
- **Refined Scraping**: Fetches high-quality, English-only reviews from Google Play Store.
- **AI Core**: Themes, quotes, and action ideas generated by Google Gemini (2.0 Flash).
- **Premium Dashboard**: INDMoney-themed Streamlit UI for data visualization.
- **Automated Pulse**: Weekly reports dispatched via SMTP with CSV attachments.
- **Scheduled Pipeline**: Pre-configured for Windows Task Scheduler.

## Local Setup

### 1. Prerequisite
- Python 3.8+
- [Google AI API Key](https://aistudio.google.com/apikey)

### 2. Installation
Clone the repository and install dependencies:
```bash
git clone https://github.com/your-username/indmoney-weekly-pulse.git
cd indmoney-weekly-pulse
pip install -r requirements.txt
```

### 3. Environment Configuration
Create a `.env` file in the root directory (see `.env.example`):
```env
GOOGLE_API_KEY=your_gemini_api_key

# Email/SMTP Settings
SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
SENDER_EMAIL=your_email@gmail.com
SENDER_PASSWORD=your_app_password
RECIPIENT_EMAIL=stakeholder_email@example.com
```
*Note: If using Gmail, use a [Google App Password](https://myaccount.google.com/apppasswords).*

## Running the Project

### Dashboard (User Interface)
```bash
streamlit run streamlit_app.py
```

### Manual Pipeline Execution
```bash
python Phase6/scheduler.py
```

### Automation (Windows)
Set up **Task Scheduler** pointing to `Phase6/scheduler.py`.

## Project Structure
- `Phase1`: Scraper logic (Google Play Store)
- `Phase2`: Gemini AI Analyzer
- `Phase3`: Legacy Dashboard files
- `Phase4`: Email Generation & Mailer
- `Phase5`: Verification Scripts
- `Phase6`: Automation Scheduler
- `streamlit_app.py`: Main deployment entry point
